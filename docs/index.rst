jupyterbook.pub documentation
=====================================================

JupyterBook.pub is a WIP tool to fetch "some content" and "do something with it". For
now, that means fetching any supported content using `repoproviders`_, and then building
a Jupyter Book project, or a JupyterLite instance.

.. _repoproviders: https://github.com/yuvipanda/repoproviders/
